# Markdown Editor 0.2.0

Esta versão melhora a navegação e permite continuar o trabalho de onde ele foi
interrompido.

## Novidades

- Use `Ctrl+G` ou **Editar > Ir para a linha...** para mover o cursor diretamente
  ao início de uma linha.
- O menu **Arquivo > Arquivos recentes** mantém até nove documentos abertos ou
  salvos recentemente.
- Ao iniciar sem informar um arquivo pela linha de comando, o editor reabre o
  documento que estava ativo ao sair.
- A última linha visitada é memorizada separadamente para cada arquivo e
  restaurada ao abri-lo pelo diálogo, pelo menu de recentes ou pela linha de
  comando.

## Instalação

Baixe e execute `markdown-editor-0.2.0-setup.exe`. O instalador permite associar
arquivos `.md` e `.markdown` e iniciar o editor ao terminar.

Para usar o programa sem instalação, baixe
`markdown-editor-0.2.0-portable.zip` e extraia a pasta `markdown-editor`.

Requer Windows de 64 bits e o Microsoft Edge WebView2 Runtime. O instalador
ainda não possui assinatura digital, portanto o Windows poderá exibir um aviso
de segurança.
